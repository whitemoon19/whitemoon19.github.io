变量类型与计算

值类型与引用类型

栈从上往下累加，堆从下往上排列

值类型的value直接存储在栈中，引用类型在堆中创建键值对，栈中存储的value为内存地址

常见值类型

*const定义常量必须有值，用let可以无值（let a）

undefined，字符串，布尔，symbol，数字 (number)

常见引用类型

对象，数组 (array)、null

typeof运算符

  • 识别所有值类型
  • 识别函数
  • 判断是否是引用类型object（不可再细分）

深拷贝

变量计算 - 类型转换

字符串拼接

==

会隐式发生类型转换，让它们尝试相等

除了判断对象是null之外用==，其他一律用 ===

obj.a==null 相当于obj.a === null || obj.a ===undefined

if语句和逻辑运算 truly和falsely变量

  • truly变量：

    !!a === true的变量。

    例如，const n =100

    !n === false，!!n == true

  • falsely变量：

    • 0
    • NaN（是number类型，指不是一个数字，例如字符串想要转换为数字）
    • ‘’
    • null
    • undefined
    • false

    除此以外都是truly变量

    逻辑判断直接判断是truly还是falsely变量来返回值
